Four design pillars

1. Deterministic Recognition

The recognition path is CTC-only. No generative model is invoked to produce characters or tokens. What the camera saw is what the pipeline writes.

  • PaddleOCR 2.9.1 with CTC decoding
  • Tesseract fallback for edge cases
  • Adaptive DPI escalation (300 / 450 / 600)
  • No LLM substitution path in core OCR

2. Forensic Image Preservation

When OCR fails on a page, the original image is preserved in the output PDF as an image-only page. Evidence is never discarded — failures become visible, not invisible.

  • Image-only fallback in final PDF
  • Page-level crash resume
  • Audit entries for every fallback
  • Output integrity validation

3. Hash-Chained Custody

Every document, every page, every transformation produces a JSONL custody event. Each event is SHA-256 hash-chained to the previous one — tamper-evident, append-only, replayable.

  • SHA-256 hash chain across the run
  • JSONL audit log (machine-friendly)
  • Replayable from any checkpoint
  • Signed evidence bundles (opt-in)

4. Defensible by Design

Privilege detection during structured extraction. NC-licensed models gated by tenant policy. Translation tier separated from forensic core. Generative enrichment is opt-in and labeled in custody.

  • Privilege keyword filtering
  • Tenant-level model gating
  • Forensic-core vs AI-adjacent boundary
  • Opt-in translation with custody events

Where EDCOCR fits

Electronic discovery

Searchable production sets with Bates stamping, chain of custody, and load-file generation.

Legal document processing

Privilege detection during structured extraction. Defensible redaction. Multi-language case files.

Forensic investigation

Tamper-evident audit trails. Replayable hash chains. Per-span language attribution.

Government & classified

Air-gapped deployment with 45 pre-baked language models. Zero internet egress required.

Healthcare records

Per-tenant isolation. PII / PHI spatial extraction. HIPAA-readiness documentation.

Insurance & claims

High-volume batch processing. Handwriting detection. Form / KV extraction.

Deployment posture

Single host

Docker Compose with NVIDIA passthrough. 5-minute install. Suits up to ~5,000 pages/day.

Distributed

Django coordinator + Celery worker fleet. PostgreSQL, RabbitMQ, Redis. Scales to 100k+ pages/day.

Kubernetes

Production Helm chart with 26 templates. KEDA autoscaling, PDBs, network policies, Prometheus stack.
